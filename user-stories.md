Guest/User Stories



View Rooms \& Availability



As a guest, I want to view available rooms so that I can choose a room for my stay.



Book a Room



As a guest, I want to book a room online so that I can reserve my stay easily.



Cancel or Modify Booking



As a guest, I want to cancel or modify my booking so that I can adjust my travel plans.



View Booking History



As a guest, I want to view my booking history so that I can track past and upcoming stays.



Online Payment



As a guest, I want to pay online so that my booking is confirmed immediately.



Request Additional Services



As a guest, I want to request additional services (like breakfast or spa) so that my stay is more convenient.



Receive Notifications



As a guest, I want to receive email/SMS notifications for booking confirmation, payment, and reminders so that I am informed.



Provide Feedback



As a guest, I want to rate and review my stay so that I can share my experience.



Multi-language Support



As a guest, I want the system to support multiple languages so that I can use it comfortably. 







Admin/Staff User Stories



Manage Room Inventory



As a hotel admin, I want to add, update, or remove rooms so that the room availability is accurate.



Manage Bookings



As a hotel admin, I want to view, confirm, modify, or cancel bookings so that reservations are handled efficiently.



Generate Reports



As a hotel admin, I want to generate reports on occupancy, revenue, and booking trends so that I can analyze business performance.



Manage Staff Accounts



As a hotel admin, I want to create and manage staff accounts with roles and permissions so that only authorized personnel can access the system.



View Guest Feedback



As a hotel admin, I want to view guest ratings and reviews so that I can improve services.



Manage Additional Services



As a hotel admin, I want to configure optional services (like spa, meals, transport) so that guests can select them during booking.
