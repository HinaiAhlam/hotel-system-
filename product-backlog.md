| ID | User Story                                                                                                | Priority | Acceptance Criteria / Notes                                                                                                        |

| -- | --------------------------------------------------------------------------------------------------------- | -------- | ---------------------------------------------------------------------------------------------------------------------------------- |

| 1  | As a guest, I want to view available rooms so that I can select a room to book.                           | High     | - Display room types, availability, pricing, amenities.<br>- Filter by date, room type, and occupancy.                             |

| 2  | As a guest, I want to book a room online so that I can reserve my stay easily.                            | High     | - Booking form with guest details.<br>- Confirm booking and provide booking ID.<br>- Payment integration optional (Stripe/PayPal). |

| 3  | As a guest, I want to cancel or modify my booking so that I can adjust my plans.                          | Medium   | - Allow cancellation or date changes before a defined deadline.<br>- Update availability accordingly.                              |

| 4  | As a guest, I want to view my booking history so that I can track past and upcoming stays.                | Medium   | - Display past and upcoming bookings.<br>- Include booking details, payment status, and room info.                                 |

| 5  | As a hotel admin, I want to manage room inventory so that I can keep availability updated.                | High     | - Add, edit, and remove rooms.<br>- Set room pricing, capacity, and amenities.                                                     |

| 6  | As a hotel admin, I want to manage bookings so that I can handle guest reservations efficiently.          | High     | - View all bookings.<br>- Confirm, modify, or cancel bookings.<br>- Track payment status.                                          |

| 7  | As a hotel admin, I want to generate reports so that I can analyze occupancy and revenue.                 | Medium   | - Reports by date, room type, occupancy rate, and revenue.<br>- Export to CSV/PDF.                                                 |

| 8  | As a guest, I want to make payments online so that my booking is confirmed instantly.                     | High     | - Integration with payment gateways.<br>- Confirmation after payment.<br>- Secure handling of card data.                           |

| 9  | As a guest, I want to request additional services (e.g., spa, breakfast) so that I can customize my stay. | Medium   | - Allow selection of optional services during booking.<br>- Update booking summary and price.                                      |

| 10 | As a hotel admin, I want to manage staff accounts so that only authorized users can access the system.    | Medium   | - Create, edit, and deactivate staff accounts.<br>- Assign roles/permissions.                                                      |

| 11 | As a guest, I want to receive email or SMS notifications about my booking so that I stay informed.        | Medium   | - Booking confirmation.<br>- Payment confirmation.<br>- Reminder before check-in.                                                  |

| 12 | As a guest, I want to rate and review my stay so that I can share feedback.                               | Low      | - Submit rating (1–5 stars) and comments.<br>- Display reviews on the website.                                                     |

| 13 | As a hotel admin, I want to view guest feedback so that I can improve services.                           | Low      | - Display guest reviews.<br>- Sort/filter by date or rating.                                                                       |

| 14 | As a guest, I want multi-language support so that I can use the system comfortably.                       | Low      | - Support for at least 2–3 languages.                                                                                              |
